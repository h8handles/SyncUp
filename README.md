# SyncUp

## Short Description
A scheduling MVP that helps groups find the best shared availability windows.
The idea literally came from a meme

## Current Features
- create group flow
- homepage UI
- template-based frontend
- database-backed groups and users
- scheduling engine work in progress

## Current Status
Work In Progress  
Early MVP baseline committed and stable

## Tech Stack
- FastAPI
- Jinja templates
- SQLAlchemy
- SQLite
- Python

## Local Run Instructions
To run the application locally, execute the following command:

`python -m uvicorn main:app --reload `
